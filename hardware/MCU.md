# WEMOS-D1-MINI

**WEMOS D1 mini - chip ESP8266EX**

- Features: WiFi
- Crystal is 26MHz

SPI flash characteristics (startup msg at 74880 bods):

- SPI Speed: 40MHz
- SPI Mode: QIO
- SPI Flash Size & Map: 32Mbit(512KB+512KB)

Arduino IDE config:

- Board: **LOLIN (WEMOS) D1 mini (clone)**
